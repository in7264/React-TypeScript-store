# React TypeScript Store

A modern e-commerce web application built with React and TypeScript.
The project simulates an online store where users can browse products, view detailed information, add items to the cart and favorites, and manage their shopping experience.

## Getting Started

### Clone the repository

```bash
git clone https://github.com/in7264/react-typescript-store.git
cd react-typescript-store
```

### Install dependencies

```bash
npm install
# or
yarn install
```

### Run the project locally

```bash
npm run dev
# or
yarn dev
```

Then open:

```
http://localhost:5173
```

## Features

* Product catalog with categories
* Product details page
* Shopping cart management
* Favorites (wishlist)
* Responsive design
* Routing between pages
* State management with Redux Toolkit
* Loading and error states
